## Dependencies
 - express
 - mongoose
 - dotenv
 - cors

## DevDependencies
  - nodemon

## Installation all dependencies

Use the package manager [npm]

```
npm i or npm install
```

## Envs

Create file [.env] in the root proyect and add...

```
NODE_ENV=local
HOST=localhost
PORT=3001
MONGODB_URI=mongodb://localhost/practicas
```
